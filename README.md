# 农业地块与农作物管理 App

基于 Flutter 实现的农业管理应用 UI，包含 5 个主要页面，已接入真实地图功能。

## 页面功能

### 1. 地图圈地与面积 ⭐ 已接入真实地图
- **OpenStreetMap 地图**：支持标准地图和卫星视图切换
- **点击描点**：在地图上点击添加边界点
- **实时面积计算**：使用 Shoelace 公式自动计算面积（平方米和亩数）
- **周长计算**：自动计算边界周长（米）
- **点位管理**：
  - 撤销最后一个点
  - 长按删除指定点
  - 清除所有点
- **地块命名**：支持编辑地块名称
- **保存功能**：保存地块信息

### 2. 作物档案
- 作物摘要卡片（编号、名称、品种）
- 核心数据网格（耕种日期、播种面积、生长阶段、采收日期）
- 地块作物记录表
- 田间操作记录（追肥、灌溉、巡检）

### 3. 气象农事提醒
- 实时天气数据（温度、降雨、湿度）
- 土壤降雨推断
- 农事提醒列表（浇水、施肥、收获）
- 智能天气摘要

### 4. AI 长势识别
- 拍照识别作物长势
- 识别结果摘要（作物、置信度、生育期）
- 生长数据网格（长势评分、叶绿素、缺水风险、病斑概率）
- 问题诊断
- AI 建议

### 5. 经营总览与任务
- 农场经营数据（耕地面积、作物批次、待办事项）
- 地块监控
- 今日农事任务列表
- 智能提醒摘要

## 运行方式

### 前置要求
- 安装 [Flutter SDK](https://flutter.dev/docs/get-started/install)（版本 >= 3.0.0）
- 配置 Flutter 环境变量

### 运行步骤

```bash
# 1. 进入项目目录
cd flutter_app

# 2. 获取依赖
flutter pub get

# 3. 运行应用
flutter run
```

### 构建发布版本

```bash
# Android APK
flutter build apk

# iOS（需要 Mac 环境）
flutter build ios

# Web
flutter build web
```

## 地图功能说明

### 使用的地图服务
- **默认地图**：OpenStreetMap（免费，无需 API Key）
- **卫星地图**：Esri World Imagery

### 地图操作方式
1. **添加边界点**：单击地图任意位置
2. **删除单个点**：长按已添加的边界点
3. **撤销操作**：点击右上角撤销按钮
4. **清除所有**：点击删除按钮（会弹出确认框）
5. **切换视图**：点击右上角"卫星"/"地图"按钮

### 面积计算原理
- 使用 **Shoelace 公式**（鞋带公式）计算多边形面积
- 根据经纬度转换为实际平方米
- 自动换算为亩（1 亩 = 666.67 平方米）
