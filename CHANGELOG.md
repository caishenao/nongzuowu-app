## [1.0.0] - 2026-06-09

### Added
- 🗺️ 地图圈地与面积计算功能
  - OpenStreetMap 真实地图集成
  - 点击添加边界点
  - 多边形绘制和面积计算
  - 支持卫星视图切换
  
- 🌱 作物档案管理
  - 作物信息展示
  - 生长阶段跟踪
  - 核心数据网格
  - 田间操作记录
  
- 🌤️ 气象农事提醒
  - 实时气象数据展示
  - 土壤湿度监测
  - 农事提醒列表
  - 智能提醒摘要
  
- 📸 AI 长势识别
  - 拍照识别作物长势
  - 生长数据展示
  - 问题诊断
  - AI 建议
  
- 📊 经营总览与任务
  - 经营数据统计
  - 地块监控
  - 今日任务列表
  - 智能提醒

### Technical
- Flutter 3.19+ 前端框架
- Spring Boot 3.3 后端服务
- PostgreSQL 数据库
- Provider 状态管理
- Dio 网络请求
- GitHub Actions CI/CD
